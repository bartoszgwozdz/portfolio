Projects

sprawdzsluch.pl

SprawdzSluch.pl is a hearing test platform built on a microservices backend architecture. The test itself is implemented using the Web Audio API, while dedicated backend services handle test result persistence, payment processing, and email notifications. This design ensures scalability, reliability, and separation of concerns, allowing each service to evolve independently while maintaining a seamless user experience.
